# vue-component-boilerplate
vue开发组件的脚手架

### Usage

```
1. npm install
2. npm run build
3. npm run dev
4. open 0.0.0.0:8082
```
  
### Features

* vue



### 目录结构
```
├── dist					// 组件打成umd格式后输出的目录
├── src 					// 组件目录
├── demo					// demo目录					
├── package.json
└── README
```

### Explation

1. npm run build： 打成umd格式的包输入到dist
2. npm run build:es6： 将src代码原样输出到dist
